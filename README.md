## 功能：

添加：可以通过模糊搜索，和精确搜索，在搜索结果里选择要添加的项目。

搜索：通过实时搜索，显示本地已存在的项目。

分类：通过标签的方式，进行分类。

## 技术栈：

```
electron + vue + nedb
```

## 构建：

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:9080
yarn dev

# build electron application for production
yarn build


# lint all JS/Vue component files in `src/`
yarn lint

```
