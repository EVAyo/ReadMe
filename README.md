<h1 align="center">README</h1>



### ✨特性

- 🍕 全平台支持. Windows, Mac, Linux
- 🍥 支持 star 管理，收藏和取消收藏
- 🍔 支持 README.md 查看
- 🍟 支持快速在线编辑代码和本地编辑器编辑代码
- 🌭 支持 md 导出图片、html
- 🍿 支持显示仓库 star 增长趋势
- 🥙 支持搜索仓库、代码、反馈、用户
- 🥪 支持 trending 趋势榜
- 🌮 支持查看 gist 代码
- 🍣 支持查看动态
- 🍤 支持多语言、多主题
- 🥮 使用 cdn 加速，解决未翻墙用户无法打开的问题
